see.term module¶

Terminal info.

see.term.line_width(default_width=78, max_width=120)¶

Return the ideal column width for the output from see.see(), taking the terminal width into account to avoid wrapping.

see.term.term_width()¶

Return the column width of the terminal, or None if it can’t be determined.
